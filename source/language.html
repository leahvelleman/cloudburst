class language.Language(key_lexicon: pynini.Fst) → None¶

Bases: language.Level

A language.

In Cloudburst a language is made up of one or more levels of representation. Forms that are identical at the key level are treated as identical for other purposes.

Parameters:key_lexicon – A Pynini acceptor that will provide a lexicon for the key level of the language.
sigma_star¶

A Pynini acceptor representing the closure over the alphabet of the language.

class language.Level(derivation: pynini.Fst, parent: typing.Union[language.Level, NoneType] = None) → None¶

Bases: object

A level of representation for language forms.

derivation¶

If this is the key level of its language, a Pynini acceptor constituting the key lexicon. If this is a child level, a pynini transducer deriving representations at this level from representations at the parent level.

parent¶

The parent level that this one is defined in terms of; None if this is the key level of its language.

children¶

A list of levels that are defined in terms of this one.

converters¶

If thislevel and otherlevel are levels in the same language, thislevel.converters[otherlevel] is a transducer converting from representations at thislevel to representations at otherlevel.

language¶

Language this level belongs to.
