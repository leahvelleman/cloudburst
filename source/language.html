class language.Level(fst: pynini.Fst, parent: typing.Union[language.Level, NoneType] = None) → None¶

Bases: object

A level of representation for language forms.

A level wraps a finite state transducer which converts keys into strings. The level provides lookup() and lookdown() functions to apply this transducer in both directions.

>>> L = Language(...)
>>> spelling = L.add_child(...)
>>> spelling.lookup("guapo")

>>> spelling.lookdown("handsome[+Adj][+Masc][+Sg]")

A level also provides convenience functions for defining new levels based on it. For instance, if you already have a level whose lookdown() function turns keys into written words, you might use it as the basis for a new level representing pronunciation.

>>> pronunciation = spelling.add_child(...)

>>> pronunciation.lookup("gera")

Levels created in this way form a tree, and strings can be converted between any two levels that belong to the same tree, using conversion functions stored in the converters attribute.

>>> pronunciation.converters[spelling]("gera")

Finally, a level can be called with a string argument to produce a Sphinx document tree node representing it. The & operator can be used as a shortcut to combine several levels of representations in a single document tree node.

>>> spelling("guerra")

>>> spelling("guerra") & pronunciation

>>> spelling & pronunciation("gera")
fst¶

If this level is a key level, fst is a Pynini acceptor representing the possible keys. If it is a derived level, fst is a Pynini transducer deriving it from its parent level.

parent¶

The parent level that this one is defined in terms of; None if this is the key level of its language.

children¶

A list of levels that are defined in terms of this one.

converters¶

If thislevel and otherlevel are levels in the same language, thislevel.converters[otherlevel] is a transducer converting from representations at thislevel to representations at otherlevel.

language¶

Language this level belongs to.

rewrite(change, left_environment='', right_environment='', direction='ltr', mode='obl') → language.Level¶

Create a level derived from this one by a context-dependent rewrite rule.

This method wraps the built-in Pynini pynini.cdrewrite() method. It provides friendlier argument names (in particular, avoiding the keyword lambda), takes care of specifying the correct sigma_star, and ensures that the result is a Level object instead of a bare Fst.

Parameters:
  • change (pynini.Fst) – A pynini transducer representing the change to be made.
  • left_environment (pynini.Fst or str) – A pynini acceptor or string representing the left environment for the change.
  • right_environment (pynini.Fst or str) – A pynini acceptor or string representing the right environment for the change.
Returns:

The newly created level.
Return type:

Level
